# MyMenu 🍽️

אפליקציית **PWA בעברית (RTL)** לתכנון ורישום תפריט יומי — אישית, חיובית ומעודדת.
כל הנתונים נשמרים **מקומית על המכשיר בלבד** (IndexedDB). אין שרת, אין רשת, אין מעקב.
האפליקציה עובדת **אופליין מלא** אחרי הטעינה הראשונה.

---

## הרצה בפיתוח

דרוש Node.js 20+.

```bash
npm install      # התקנת תלויות (פעם אחת)
npm run dev      # שרת פיתוח על http://localhost:5173
```

## בנייה ל-production

```bash
npm run build    # בונה ל-dist/ כולל Service Worker ו-manifest (PWA)
npm run preview  # תצוגה מקומית של ה-build על http://localhost:4173
```

## בדיקות

```bash
npm run test      # בדיקות יחידה (Vitest) — מנוע ההמלצות, הניקוד, אימות ועוד
npm run test:e2e  # בדיקות קצה-לקצה (Playwright) על viewport מובייל 390×844
npm run icons     # יצירה מחדש של אייקוני ה-PWA מתוך public/icons/icon.svg
```

> בדיקות ה-E2E דורשות דפדפן Playwright. אם עדיין לא הותקן:
> `npx playwright install chromium`

## התקנה על הטלפון (PWA)

הפעילו קודם `npm run build && npm run preview` (או פרסמו את תיקיית `dist/` לאירוח כלשהו),
וגלשו לכתובת מהטלפון. אז:

### אנדרואיד (Chrome)
1. פותחים את האתר ב-Chrome.
2. תפריט ⋮ (שלוש נקודות) → **"הוספה למסך הבית"** / *Add to Home screen*.
3. מאשרים — צלמית MyMenu מופיעה במסך הבית ונפתחת כאפליקציה מלאה.

### אייפון / אייפד (Safari)
1. פותחים את האתר ב-Safari.
2. לוחצים על כפתור **שיתוף** (הריבוע עם החץ למעלה).
3. גוללים ובוחרים **"הוספה למסך הבית"** / *Add to Home Screen* → **הוסף**.
4. הצלמית מופיעה במסך הבית ונפתחת במסך מלא.

לאחר ההתקנה האפליקציה **עובדת גם ללא אינטרנט** — כל המידע נשמר על המכשיר.

## עקרונות המוצר

- **עברית בלבד** בממשק, כיווניות RTL מלאה.
- **חוויה חיובית בלבד**: אין קלוריות, אין משקל, אין דיאטה — רק עידוד וגיוון.
  הממתק היומי הוא חלק מתוכנן ולגיטימי מהתפריט.
- **תזונה מאוזנת לילדים**: חלבון בכל ארוחה עיקרית (בוקר/צהריים/ערב),
  לפחות 2 מנות ירקות ו-2 מנות פירות ביום, כיסוי כל קבוצות המזון, ומד מים.
- **מנוע לומד**: דירוגי טעם ושובע משפיעים על ההמלצות למחר; מנה עיקרית לא חוזרת בטווח 3 ימים.
